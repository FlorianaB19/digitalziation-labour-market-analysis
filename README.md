📊 Digitalization and Labor Market Analysis

This project is an interactive data analysis dashboard built with Python and Streamlit, designed to explore the impact of digitalization on the labor market across 27 European countries over the period 2007–2022.

🎯 Project Overview

The application integrates data analysis, visualization, machine learning, and clustering techniques to examine how digital transformation influences employment dynamics.

It allows users to:

explore economic and digital indicators
compare countries and trends over time
analyze relationships between variables
build predictive models
identify patterns using clustering
📁 Dataset
Source: World Bank Open Data
Type: Panel data (country-year)
Coverage:
🌍 27 European countries
📅 Period: 2007–2022
Includes indicators such as:
Employment rate
Internet usage
ICT goods exports
GDP per capita
R&D expenditure
Broadband subscriptions
⚙️ Features
📈 Exploratory Data Analysis
Interactive filtering by country and time period
Descriptive statistics
Time-series visualization of indicators
🌍 Geographic Visualization
Choropleth maps of Europe
Country-level comparison of indicators
📊 Correlation Analysis
Correlation matrix between economic and digital variables
🤖 Machine Learning
Models implemented:
Linear Regression
Decision Tree Regressor
Random Forest Regressor
Performance metrics:
MAE, RMSE, R²
MAPE, MASE
Residual diagnostics using ACF
Feature importance analysis
🧩 Clustering Analysis
K-Means clustering applied for years:
2007, 2013, 2022
Identification of groups of countries with similar digital profiles
Visualization of clusters
🛠️ Technologies Used
Python
Streamlit
Pandas & NumPy
Plotly
Scikit-learn
Statsmodels



🎓 Purpose

This project was developed as part of a Bachelor’s thesis and aims to provide both:

a data-driven perspective on digitalization and employment
an interactive analytical tool for exploring economic and digital indicators
📌 Key Insights

The analysis highlights:

strong relationships between digitalization and employment
significant differences across European countries
the importance of digital infrastructure and innovation
the potential of machine learning in economic analysis
